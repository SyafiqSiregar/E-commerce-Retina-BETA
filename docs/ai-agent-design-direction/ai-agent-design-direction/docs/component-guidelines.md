# Component Guidelines

## Global Component Rules

Semua komponen harus mengikuti aturan:
- Border-driven.
- Minimal shadow.
- Rounded corner besar.
- Typography compact.
- Warna netral dominan.
- Violet hanya untuk emphasis utama.

## Header

Header harus:
- Putih.
- Border bawah hairline.
- Sticky optional.
- Navigasi compact.
- Logo/brand boleh memakai violet.
- CTA kanan boleh memakai outline atau violet jika benar-benar primary.

Desktop:
- Logo kiri.
- Nav tengah atau kanan.
- CTA kanan.

Mobile:
- Logo kiri.
- Menu button kanan.
- Dropdown sederhana.

## Hero Section

Hero harus:
- Memiliki headline pendek.
- Subheadline ringkas.
- CTA utama.
- Visual produk atau card preview.
- Tidak menggunakan gradient dekoratif.
- Background putih atau soft canvas.

Headline boleh lebih besar dari type scale normal, tetapi tetap clean.

## Search Bar

Search bar:
- Background putih.
- Border hairline.
- Radius 24–28px.
- Input text 14px.
- Submit button violet kecil atau icon arrow violet.
- Placeholder warna concrete.

## Product Card

Product card:
- Border hairline.
- Radius 24px.
- Background white.
- Image area dominan.
- Title 14px.
- Metadata 11–12px.
- Rating kecil jika diperlukan.
- Harga atau status compact.

Hover:
- Border menjadi lebih gelap.
- Shadow sangat halus optional.
- Tidak perlu transform berlebihan.

## Feature Card

Feature card:
- Background white atau soft canvas.
- Border hairline.
- Radius 24px.
- Icon hitam/slate.
- Text compact.
- Violet hanya jika feature sangat penting.

## Badge

Badge:
- Small.
- Border hairline.
- Text 10–11px.
- Radius 999px boleh untuk badge kecil.
- Background putih atau soft canvas.
- Hindari banyak warna badge.

## CTA Section

CTA section:
- Minimal.
- Copy pendek.
- Satu primary action.
- Optional secondary action.
- Background white atau near-black jika ingin kontras.
- Jika dark, gunakan white text dan border subtle.

## Footer

Footer:
- Compact.
- Banyak neutral text.
- Border top hairline.
- Link kecil 11–12px.
- Tidak perlu warna dekoratif.
