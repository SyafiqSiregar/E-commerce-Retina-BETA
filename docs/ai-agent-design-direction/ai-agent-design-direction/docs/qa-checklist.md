# QA Checklist

## Visual QA

- [ ] Canvas utama putih.
- [ ] Warna violet hanya dipakai pada elemen penting.
- [ ] Tidak ada gradient dekoratif.
- [ ] Tidak ada shadow berlebihan.
- [ ] Border terlihat tipis dan konsisten.
- [ ] Card memakai radius 22–28px.
- [ ] Typography compact.
- [ ] Layout terasa rapi dan premium.
- [ ] Product/image menjadi fokus visual.

## Responsive QA

- [ ] Mobile layout rapi.
- [ ] Tablet layout rapi.
- [ ] Desktop layout rapi.
- [ ] Tidak ada overflow horizontal.
- [ ] Header mobile mudah digunakan.
- [ ] CTA tetap terlihat jelas.

## Accessibility QA

- [ ] Semua image punya alt text.
- [ ] Semua button bisa diakses keyboard.
- [ ] Focus state terlihat.
- [ ] Contrast text cukup.
- [ ] Semantic HTML digunakan.
- [ ] Form input punya label atau aria-label.

## Code QA

- [ ] Komponen reusable.
- [ ] Tidak ada duplikasi class berlebihan.
- [ ] Token warna digunakan konsisten.
- [ ] File structure rapi.
- [ ] Tidak ada dependency tidak perlu.
- [ ] Tidak ada console log debug.
