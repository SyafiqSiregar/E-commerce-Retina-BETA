# Build Agent Prompt

You are building a modern product-first website using Next.js, TypeScript, and Tailwind CSS.

Follow these documents:
- `docs/design.md`
- `docs/agent.md`
- `docs/component-guidelines.md`
- `docs/page-structure.md`
- `docs/interaction-guidelines.md`
- `docs/qa-checklist.md`

Your goal is to create a clean near-monochrome website inspired by a premium commerce interface.

## Mandatory Design Rules

- Use white as the main canvas.
- Use `#5433eb` only for important primary actions.
- Use thin borders instead of heavy shadows.
- Use compact typography.
- Use rounded cards around 22–28px.
- Avoid decorative gradients.
- Avoid colorful UI unless it comes from product imagery.
- Keep layout minimal and structured.

## Build Order

1. Create design tokens in global CSS.
2. Configure Tailwind theme if needed.
3. Create base layout.
4. Create reusable UI components.
5. Create homepage sections.
6. Make responsive.
7. Run QA checklist.
8. Refactor duplicated styles.
