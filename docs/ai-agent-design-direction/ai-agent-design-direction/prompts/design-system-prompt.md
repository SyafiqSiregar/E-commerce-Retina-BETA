# Design System Prompt

Create a design system for this project based on these principles:

- White-first canvas.
- Near-monochrome UI.
- Violet primary accent: `#5433eb`.
- Hairline border: `#ebebeb`.
- Text black: `#000000`.
- Secondary text: `#707070`.
- Soft surface: `#f2f4f5`.
- Compact typography.
- Rounded cards.
- Minimal shadows.

Generate:
1. Color tokens.
2. Typography tokens.
3. Spacing tokens.
4. Radius tokens.
5. Button variants.
6. Card variants.
7. Form input variants.
8. Badge variants.
9. Layout primitives.

Do not create a colorful design system. Keep it minimal and product-focused.
