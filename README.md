# 🛡️ Retina Security Technology - E-commerce

Website E-commerce modern untuk solusi keamanan CCTV, terintegrasi langsung dengan database Google Sheets untuk manajemen stok secara real-time.

[![GitHub Status](https://img.shields.io/badge/Status-BETA-orange.svg)](https://github.com/SyafiqSiregar/E-commerce-Retina-BETA)
[![Vue Version](https://img.shields.io/badge/Vue-3.x-4fc08d.svg)](https://vuejs.org/)
[![Vite](https://img.shields.io/badge/Vite-Built-646cff.svg)](https://vitejs.dev/)

## 🚀 Fitur Utama
- **Real-time Inventory**: Sinkronisasi data stok dan harga langsung dari Google Sheets.
- **Dynamic Category**: Filter produk berdasarkan kategori secara otomatis.
- **WhatsApp Checkout**: Integrasi pemesanan langsung ke WhatsApp penjual.
- **Responsive Design**: Tampilan premium yang optimal di desktop maupun mobile.
- **Professional Aesthetics**: Menggunakan desain modern dengan micro-animations dan glassmorphism.

## 🛠️ Tech Stack
- **Frontend**: Vue.js 3 (Composition API)
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom UI System)
- **Database**: Google Sheets API (via CSV/JSON Export)
- **Routing**: Vue Router

## 📂 Struktur Proyek
- `/retinav2-vue`: Folder utama aplikasi Vue.
- `/js`: Script legacy/utilitas untuk integrasi.
- `/css`: Stylesheets pendukung.
- `/public/images/products`: Lokasi penyimpanan foto produk.

## ⚙️ Cara Menjalankan Lokal
1. Masuk ke folder vue:
   ```bash
   cd retinav2-vue
   ```
2. Install dependensi:
   ```bash
   npm install
   ```
3. Jalankan server development:
   ```bash
   npm run dev
   ```
